AI Social Media Automation Bot

Telegram-бот для генерации, редактирования и публикации постов
в Telegram, Instagram и Facebook с использованием ИИ.

Функции:
- Генерация текста
- Генерация изображений
- Редактирование
- Мультиязычность
- Кросс-постинг

Запуск:
pip install -r requirements.txt
python bot.py
